# C-Language-Learnings

This repository serves as a digital archive of my early programming journey. It contains the foundational code files, exercises, and mini-projects I completed while mastering the C programming language.

## 🚀 About This Repository
This collection documents my transition from a beginner to a more confident coder. It covers the essential building blocks of software development, focusing on the core principles of C. Every file here represents a step in my learning process, ranging from syntax basics to more complex logic.

## 📂 Repository Contents
The code is organized to reflect the progression of my learning path:

* **Fundamentals:** Basic variable declarations, data types, and arithmetic operations.
* **Control Flow:** Implementation of loops (`for`, `while`, `do-while`) and conditional statements (`if-else`, `switch`).
* **Advanced Logic:** Star patterns, complex loop structures, and algorithmic problem-solving.
* **Functions & Recursion:** Understanding modular programming, function calls, and the call stack.
* **Memory Management:** Pointers, arrays, and dynamic memory allocation concepts.
* **File Handling:** Reading from and writing to files to handle persistent data.
* **Mini-Projects:** Small-scale applications that combine multiple concepts into functional programs.

## 🛠 Skills Developed
Through the creation of these files, I developed a strong grasp of:
* **Memory management** and pointers.
* **Systematic debugging** and logical problem-solving.
* **Algorithmic thinking**, specifically translating mathematical concepts into executable code (e.g., recursive algorithms).
* **Code structure**, emphasizing modularity and readability.

## 📈 Current Status
While this repository focuses on my "initial phase," it remains a key reference point for my foundational knowledge. I have since expanded my focus into modern domains, including Generative AI and agentic frameworks, but this repository remains the bedrock of my technical background.
